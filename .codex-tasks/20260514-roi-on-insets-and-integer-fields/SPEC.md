# Task Specification

## Task Shape

- **Shape**: `single-full`

## Goals

- Remove the ROI-to-derived Source Image crop action from the UI/controller.
- Allow ROI drawing on ROI-generated inset images, not only original Source Image objects.
- Make right-side selection bounds fields integer-only.

## Non-Goals

- Do not remove project JSON compatibility for previously saved derived Source Images.
- Do not add recursive/cascading delete behavior for parent ROIs.
- Do not change export behavior.

## Constraints

- React + TypeScript + Konva.
- Follow existing reducer/action patterns.
- Keep source files under 300 lines and production functions under 50 lines.
- Preserve visible errors for invalid graph operations.

## Environment

- **Project root**: `D:\code\Figure Composer`
- **Language/runtime**: TypeScript / React
- **Package manager**: npm
- **Test framework**: Vitest
- **Build command**: `npm.cmd run build`
- **Existing test count**: 35 tests before this task

## Risk Assessment

- [x] External dependencies unavailable or unnecessary.
- [x] Breaking changes assessed through reducer/project reference tests.
- [x] Large file generation not involved.
- [x] Backend unit tests are not involved.

## Deliverables

- Derived crop action removed from app UI/controller.
- ROI coordinate mapping supports source image objects and inset objects.
- Right selection panel bounds inputs display and submit integers.
- Automated validation passes.

## Done-When

- [x] Inspector no longer offers Crop Source / derived Source Image creation.
- [x] A new ROI can be created from an inset object generated by an earlier ROI.
- [x] Deleting a parent ROI with child ROIs fails before invalid references are created.
- [x] Project reference validation accepts ROI source objects that are insets.
- [x] Selection panel x/y/width/height fields use integer display and integer patches.
- [x] `npm.cmd run test`, `npm.cmd run typecheck`, and `npm.cmd run build` pass.

## Final Validation Command

```powershell
npm.cmd run test
npm.cmd run typecheck
npm.cmd run build
```

## Demo Flow

1. Import an image.
2. Draw an ROI to create an inset.
3. Switch back to ROI tool and draw another ROI on the inset.
4. Select an object and edit bounds in the right panel with integer values.
